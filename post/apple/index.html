Apple有哪些黑科技

2020-02-22

文章目录

先看个故事，摘自知乎

某南方代工厂用的一款重要的双面胶经常出问题，一调查，发现仓库里有老鼠。老鼠不知道为啥特别中意这款双面胶，存货经常被咬的很惨，只能报废。

本来以为只是仓储的事情代工厂自己搞定就好，没想到类似的事情一而再再而三发生，一调查，仓库除鼠是世界性难题（好吧，至少在中国还没有好的决方案，仓库在建的时候就不是全封闭的）。

看代工厂搞不定，然后苹果介入了，找了一个实习生安排解决这个问题，实习生制定了一套方案，方案的核心很简单，在仓库派人养猫。

看过这个方案的人第一反应都是，不是在搞笑吧？！

详细规定了用什么猫，多大年龄，品种是什么，养几只，采购预算多少，采购途径怎么走，如何考核猫的绩效，退役年龄多大。

还规定了猫的各项福利，吃喝拉撒加退体都包含在内，养猫人的工资算做产品的成本。猫怎么样才能保持好心态，积极主动工作都考虑在内。

苹果的黑科技就是注重细节，后来这个仓库的鼠患彻底没了

跟很多硅谷公司一样，苹果常年会招聘一批实习生工作，这些人都是斯坦福，加州伯克利，MIT这些名校的硕士博士，都是学霸中的学霸。实习生来到公司以后，常常会安排一个并不迫切需要的课题，让他自由发挥，试一试做下去能做多深。

大概 06,07 年的时候，有个实习生来到苹果的 Mac PD 组，当时的 macbook 还是用塑料壳，给他的课题是研究一下，如果纯用铝cnc加工来做笔记本外壳，可行性如何。几个月后，这个实习生拿出来非常接近量产的 unibody 铝壳样品。轻巧结实，美观，直接把大家镇住了。随后乔帮主拍板做unibody，也就有了后来的 macbok air。间接影响了后面的 iphone, ipad 的设计。

这个人里我知道的最牛的实习生，名叫 Keith，结束实习后顺利留在苹果。现在用的新一代的键盘也是他设计的。如果你打开苹果网站 New Macbook 的宣传视频，在介绍 butterfly 键盘的时候，键盘上模拟有人按键，键帽依次打下来的字是：Hello Keith.

说到蝶式键盘，我的macbook曾2次因按键滞留拿去维修，苹果也承认这个缺陷，将键盘的保修延长到了4年，其中罪魁祸首是灰尘，我不得不使用键盘膜，但实际上你也可以故意进灰让后让他们免费帮你清洁主板hhh。尽管如此，我非常喜欢蝶式键盘，它美观、键程短、面积大、按键稳定，可以降低机身厚度，提高打字准确度，至于手感，因人而异，喜欢打字的请用thinkpad或是外接键盘，很多人为了黑而黑，比如ipad本身就不是为了取代电脑，有办公需求直接买电脑不就行了，苹果在键盘方面做了很多改进，但在3年后还是放弃了蝶式键盘，在新macbook16寸上回归了剪刀式键盘和实体esc键，方向键也改为到三件，我觉得很难看，还是旧的蝶式键盘+touch bar美观

说起售后，体验非常好，前提是你要去有授权的，曾经我的iPad Pro才买1个月屏幕被我摔了，一查发现要4000软妹币，整个人都不好了，后来通过某些方式补买了Apple Care +，2年内有2次意外险，花1000可以直接换个新的（不推荐效仿）。其实苹果在售后方面很宽松，以前被偷了都可以保修，但因为国人实在是太贼了，苹果不得不采取特殊政策。

现在 macbook 上用的 usb type c 是纯正的苹果设计（即为 MacBook 设计的 Thunder bold 3）。但为了向市场更快普及，借助 intel 以 usb type c 的名义发布。设计这款接口的团队也设计了 lightning, magsafe, thunder bolt 这些非常优秀的接口。

type-c非常好，很期待有那么一天所有接口被统一，生活会变得简洁而方便。反观Surface，使用了3代一样的模具，迟迟不上type-c（直到2019），坚守着自家的surface connect和无用的miniDP接口，意味这你想要使用其他接口就只能用他家的扩展坞，苹果在2016macbook上就放弃了其他接口全部使用type-c，一个过于激进，一个过于保守，苹果是为了加快市场普及，微软就是为了钱。

好在微软2018年终于识到了这个问题 虽然大部分人不会拆看机器，也不会在意主板是绿的还是黑的，但是有些东西它确实存在。

再谈lighting接口，是完全不输type-c的，它公口在外，体积更小，但容易弄脏。苹果开发了Mifi标准，一是有利可图，二是统一标准，防止劣质仿造品。在新款iPad Pro上使用type-c替换了lighting，使其扩展性更强，我很希望某一天iPhone也可以用上type-c，但这是不可能的，一方面苹果不可能放弃这么大的利润，二是苹果的目的并不只是为了让type-c代替type-A，而是要彻底砍掉接口，请参考新MacBook 12英寸标志着苹果PC时代的终结，从iPhone7砍掉耳机接口也可以看出来，苹果想构建一个以icloud为核心的无线的世界，也就是说，我们将来看到的会是一个没有任何接口的iPhone。

科技类新闻网站往往喜欢报道一类新闻：苹果最近申请了什么什么技术的新专利，然后据此预测下一代的产品会有什么大动作，云云。

实际上，所谓的专利，本质上是以公开换保护，在专利认证过程中技术细节也会一并公开。然而苹果最看重的是封闭跟保密，对于将要使用的技术永远不会申请专利（等于在上市之前公开设计细节。）而申请专利的技术都是把目前不会使用东西在法律上占个坑。因此可以总结出两个凡是： 凡是能在专利网站上看到的苹果申请专利的东西，平果一代产品都不会应用。

凡事苹果认为牛逼的技术，在技术壁垒随时间消失之前，苹果都不会拿去申请专利。

至于有人质疑，如果自己不申请专利，导致专利被人抢注了怎么办？在专利法中有一条原则，如果使用专利的一方可以证明这项技术在你申请专利之前就已经在使用了，那么可以要求法院判定你的专利无效（新颖性原则）。举个例子：你没办法在一家饭店吃到一个好吃的酱料，然后把配方注册成专利，然后反过来要求这家饭店缴纳专利费。

此原则同样适用于外观专利，中国公司可以抢在苹果前面在国内注册新 iPhone 的外观专利，但是苹果只要拿出早先研发阶段的证据证明原型机早于抢注，就可以申请被抢注的专利无效。

很多人说苹果失去了创新，不如看看隔壁的Intel，14nm挤了4年牙膏，微软已经在Suface pro x上使用了自研的ARM处理器，苹果预计2021也会在macbook上使用自研处理器，狠狠地打了Intel的脸，只能说 AMD YES！

话说回来，苹果不会在当前产品上使用最新的技术，原理很多，最重要的是保证良好的用户体验，比如柔性屏、屏下指纹，屏内挖孔，都会改变用户的使用习惯和要求开发人员重新适配，又有多少人买的起呢，至于全面刘海屏屏我也不太喜欢，但其他厂家又竞相模仿，iPhone5s刚推出指纹识别时又有哪些厂家有这个功能呢？只能苹果在创新、收益间找到了一个平衡。

苹果新出了一本产品图册《design by apple in California》，定价小两千。知乎上一堆人骂，自己的广告册还好意思卖这么贵。

我想讲的是，就算印本书，也体现了苹果的黑科技（=砸钱）。

不提原材有多贵，单讲印刷厂。这本书的代工厂是深圳雅昌，一家极其高大上的的艺术品中心，他们的主营业务是给各类著名艺术家，画家印刷制作作品图册，通常的一个产品的印刷术也就三五百本。他们是搞艺术的不屑于做量产，也不把钱看在眼里，不愿给苹果做——因为苹果的量太大，严重占用了他们的产能。

所谓产能，就是高精密的印刷机，全国除了他们家应该没有其他商业组织有了（待考）。这种印刷机精度级别可以印钞票，因此需要接受政府登记监管，数量也十分有限，不能随意扩大产能。

苹果看中了这几台设备，死磨硬泡要雅昌代工图册，说起来很蛋疼，据我所知这是第一次苹果求供应商办事。最后之所以能搞定，还是砸钱砸出来的：单本的出厂价我就不透露了，这么说吧，以这个成本，去掉税费，苹果的这批图册，卖一本赔一本。

即便如此，雅昌还是很后悔接了这个单。前面说过，这个艺术公司根本不是为了赚钱作为目标（苹果最怕这种公司）。他们为苹果制作图册，每生产一万本（具体多少不能讲）就意味着要拒绝几十单老客户的需求，让大牌艺术家为电子产品图册让路，得罪了关系户，得不偿失。

所以第一批订单搞完以后，苹果想要继续追加产能，雅昌直接拒绝，现在还在谈判。所以想买这图册的人赶紧下手吧，搞不好就是绝版。

很幸运的是，我有这本书

苹果充电线因为容易破损，被无数人吐槽。之所以做不到很结实，主要原因在于原材料要同时满足：洁白颜色（外观），不含卤素（环保），高防火等级（安全）三个要求。其实只要放松任何一个的要求，都可以做的很结实。现在的线材是满足条件下强度最好的材料。

另外、在苹果产品定义里，线材属于消耗品，你能一年换一件衬衫，一年换个充电线有什么不妥？

有一说一，线材确实容易坏，但你看看type-c的线材，对比blekin等，发现苹果竟然是最便宜的

其他的就不吹了，现在苹果公司市值已经超过1万亿了，归根到底它还是个商业公司，但不像某些大公司会阻碍科技进步（猜猜是谁），苹果始终是为了给用户提供更好的体验，国内的厂家还有很长的路要走，经济基础决定上层建筑。
