机器学习

2020-03-07

这几个星期机器学习的总结，不涉及具体代码，看了不少书，但还没自己动手写过。涉及到的概念和知识点很多，每一个部分都有很多理论支持，这里就不具体展开了(有些LateX公式之后补上)，文章末尾附有常见术语。

虽然实际应用的时候也就一行代码(框架中已经封装好了)，但了解原理有助于调整合适的参数，遇到一些实在复杂的(比如反向传播)知道作用就行，实际上很少会接触算法，大部分时候在搜集处理数据，调整参数。

其实不太推荐看书，因为技术变化得太快了，纸上的内容还停留在以前，英语好的可以直接看官方教程(可能不好懂)，而且书又贵又重又占位置，看电子版也是个不错的选择。

机器学习的一般步骤

  1. 数据预处理

    筛选有价值的数据，修复存在部分缺失的数据，通常使用留出验证将数据分为训练集和测试集，通常比例为3:1。如果是参加竞赛(如Kaggle)或是解决未知问题，则无法得到测试集，需要自己在训练集中划分出测试集(还有一种方式叫交叉验证)，在真实的机器学习应用中，大部分时间都在进行数据预处理。

  2. 选择适当的学习模型，根据经验调整参数，应用于训练集，得到预期结果

  3. 将预期结果与真实结果比较，根据损失函数判断学习性能，通常的损失函数有平均绝对误差(Mean Absolute Error, MAE)、均方误差(Mean Squared Error, MSE)

  4. 优化参数，使评价函数的结果尽可能小,通常的方式为梯度下降(gradient decent)

  5. 当训练次数达到一定数量时，损失函数几乎不再变化，可以认为训练已经结束。再将训练好的模型应用于测试集上并观察结果(通常与在训练集上的结果不一样，因为会出现过拟合和欠拟合的情况)，重复2-4步，直到得到满意结果

监督学习(Supervised Learning)

给定一组样本（通常由人工标注），它可以学会将输入数据映射到已知目标

分类学习

这里讨论最基础的二分类

经典例子：

  • 良、恶性肿瘤预测
  • 泰坦尼克号乘客是否生还

线性分类器(Linear Classifiers)

假设特征与分类结果存在线性关系，通过一条线划分数据

如果我们想要把结果映射到(0,1)，可以使用Logistic函数，将结果大于０的划分为一类

支持向量机(Support Vector Machine)

可以发现决定直线位置的并不是所以的数据，而是两个类别中间隔最小的几个数据点，称之为支持向量(Support Vector)。而在上面Logistic模型中考虑了所有数据对参数的影响，因此不一定得到最佳结果。

朴素贝叶斯(Naive Bayes)

基于概率的分类器，很简单，学过概率论的都会，但解释起来篇幅有点长

K近邻(K-Nearest Neighbor, KNN)

在特征空间中，如果一个样本附近的k个最近(即特征空间中最邻近)样本的大多数属于某一个类别，则该样本也属于这个类别。

其他

  • 决策树
  • 随机森林

回归预测

待预测的目标是连续变量，比如在训练集中有很多点(x,y)，选择一个函数使其在坐标系上的曲线尽量接近所有数据点，针对测试集中某个未知点的x值预测y值。但在实际运用中，参数是多元的、非线性可分的，需要用到数据降维

经典例子

  • 波士顿房价预测

线性回归器

类似于最小二乘法，寻找一条直线使其尽量接近所有数据

支持向量机

同样，只从训练数据中选取一部分更加有效的支持向量，不是它们的结果不再是类别目标，而是具体的预测数值

Ｋ近邻

同样只借助周围Ｋ个最近训练样本的目标数值，对待测样本的回归值进行决策

无监督学习(Unsupervised Learning)

相比与监督学习，他不需要人为得标注数据结果(如是否患病、房价数值)，因此它所应用的领域也与监督学习的不同。

聚类

对于分布在空间的很多数据，通过某些参数将它们分类

K均值(K-means)

  1. 随机设置Ｋ个特征空间内的点作为初始的聚类中心
  2. 对于每个数据，从Ｋ个聚类中心中寻找最近的一个，并且标记为从属该聚类
  3. 在所有数据都被标记后，根据这些数据新分配的类簇，重新对Ｋ个聚类中心做计算
  4. 如果一轮下来，所有数据从属的聚类中心与上一次分配的类簇没有变化，可以停止迭代，否则回到第2步

降维

在实际项目中，经常会遇到特征维度非常高的训练样本(单个训练数据包含的参数非常多)，特征降维可以重构有效的低纬度特征向量，比如使彩色照片变成黑白照片

主成分分析(Principal Component Analysis, PCA)

有点复杂。。。

神经网络(Neural network)

神经元

有多个输入和一个输出，每个层由多个神经元，一个神经元可以接受多个其他层的神经元作为输入

前向传播

给定输入参数，不同层间连接关系的权重（这里为线性关系），得到输出结果

输入层

该层的神经元没有输入，数量为单个数据特征向量的长度

隐藏层

一个神经网络有一个或多个层，层间通常为全连接

输出层

该层拥有一个或多个神经元，可以通过某些函数将输出值映射到(0,1)上，若是分类问题，神经元的个数为分类的个数，通常先使用softmax层将输出转化为概率分布

深度学习(Deep learning)

通常深度学习指的就是深度神经网络

特点

  1. 具有多层隐藏层
  2. 非线性

我们可以发现，如果使用线性函数表达每层之间的数据传递，任意多个线性模型的组合仍然是线性模型，多层和单层没有什么区别，能解决的问题非常有限，所以会使用激活函数去线性化

反向传播(backpropagation)

前面的正向传播是通过输入参数和权重得到训练结果，反向传播则是通过训练结果和预期结果，调整原先的权重。

原理：在所有参数上使用梯度下降算法，实际非常复杂

卷积神经网络(Convent neural networl, CNN)

当全连接神经网络的参数太多时，除了导致计算速度减慢，还容易导致过拟合的问题，所以需要减少神经网络中的参数个数

经典例子：MNIST手写数字数据集

输入层

比如用三维矩阵代表一张图片，长和宽分别是图像的大小，如28*28，深度代表图像的色彩，黑白图片的深度为１，RGB色彩下的深度为３

卷积层

每一个节点的输入是上一层的一小块，通常有33或55，一般通过卷积层的节点居住深度增加

池化层

不会改变三维矩阵的深度，但会减小长宽，可以理解为将高分辨率的照片转化为低分辨率的照片，从而减少整个神经网络中的参数

经典卷积神经网络

LeNet-5

Inception-v3

与上面不同的是，它同时所有不同尺寸的过滤器，然后再将得到的矩阵拼接起来

循环神经网络(Recurrent neural network, RNN)

神经网络的输出不仅取决与当前时刻的状态，还取决于之前时刻的状态

例子：自然语言处理（Natural language process, NLP），对某个单词的处理还取决于它之前的单词

长短时记忆网络(Long short-term memory, LTSM)

当模型要记录大量上下文信息时，需要根据间隔、长短调整权重，选择正确的信息，最大的特点是加入了遗忘门，让RNN“忘记”之前没用的上下文信息。

LTSM结构可以更加有效的决定哪些信息应该被遗忘，哪些信息应该得到保留。

双向循环神经网络

不仅取决于之前出现的单词，还取决于之后的单词（预扫描）

附：常见术语

激活函数

使线性函数去线性化（很奇怪的表达），将一定范围内的值映射到指定范围

损失函数

用于刻画训练结果和真实数据结果的差距，也就是评测模型的性能

通常有

  • 平均绝对误差(Mean Absolute Error, MAE)
  • 均方误差(Mean Squared Error, MSE)
  • 交叉熵(cross entropy) 刻画两个概率分布之间的距离，适用于分类问题，多用于神经网络，通常先使用Softmax回归将神经网络的输出变成一个概率分布。

过拟合

计算方法里有讲，过拟合的模型在训练集上会得到较高的正确率，但我们的目的是预测未知数据，所以还要看在测试集上的表现。

正则化

目的：避免过拟合，提高模型在未知测试数据上的泛化能力

原理：限制高次方的参数大小

  • L1范数正则化
  • L2范数正则化

梯度下降

作用：优化单个参数的取值

原理：类似于每次改变一个与导数相关的值（称之为学习率），使损失函数最小（此时最低点的导数为０），因为导数会随着接近最低点（可能是局部）而自动减小，所以不需要人为改变

学习率

控制每次参数更新的速度，需要结合梯度下降，该值小于１，通常采用指数递减的方式。
