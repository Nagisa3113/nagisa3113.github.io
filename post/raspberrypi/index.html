树莓派折腾

树莓派（raspberriPi）是一个微型的，信用卡大小的可以运行Linux的单板机。

它使用的是基于精简指令集（RISC）的ARM架构的处理器，相比于Intel采用的x86的复杂指令集（CISC），它的结构简单，功耗低，价格低，非常适合(穷)学生。

这学习开了一门《嵌入式系统导论》，老师以划水著名，像我这种学生其实完全不学的，但刚好轮到我做这门课的实验，于是我借了块板子，开始了与树莓派的奇妙冒险（最后这门课只得了70分，史上最低hhhh），实验内容主要是使用GPIO控制传感器，电子电气的同学肯定很熟悉，但对于学计算机的同学就有另一套玩法了，推荐大家都去试试，新款4b型号2g内存版本只要270，可以折腾的事情非常多！

比如说之前心心念念的GameShell，一个模块化的开源掌机，但它太贵了（其实我用的Pi 3B是嫖的同学的），然而现在你可以单独购买电池、屏幕、扬声器手柄等配件，制作属于自己的掌机，还可以用来学习操作系统和计算机网络等，总之就是非常适合用来学习。

系统安装

首先需要一个SD卡格式化软件SDFormatter

在每次安装新系统前都需要格式化

绝大多数问题都出在供电不足，树莓派4B承认供电存在问题（最新版已修复）。所以，最好的办法是使用官方的充电设备，不要使用移动电源和电脑端的USB口。

NOOBS

下载地址

即New Out of Box System（全新开箱即用系统），你只需要去官网下载然后拷贝到SD卡就可以了

优点：

  • 简单
  • 可以安装多系统

缺点

  • 体积大
  • 安装过程慢

手动烧录系统

下载地址

可以使用一键烧录软件如win32diskimager和ApplePi-Baker

这里使用MacOS/Linux下的dd命令来烧录

  1. 查看当前哪些设备已挂载，找到SD卡，如/dev/sda1

    1

    df -h

  2. 卸载设备

    1

    umount /dev/sda1

  3. 烧录，PATH为镜像的路径，bs表示一次写入多大的块，一般耗时几分钟不等

    1

    sudo dd if=PATH.img of=/dev/sda1 bs=4m; sync

开机启动

如果有显示器，直接连上就可以开机了，图形界面谁不爱呢?

如果没有，使用SSH连接，树莓派为了安全考虑默认关闭了SSH，需要你在根目录下新建一个名为SSH（没有扩展名）的文件

推荐使用网线直接连上树莓派，这里吐槽一下渣科的校园网，你需要一台可以破解校园网的路由器，虽然你可以把树莓派改造成路由器，但前提你要先有一个路由器，于是我又买了个路由器，这样一来方便多了，可惜在学校的日子也不多了。

  1. 打开命令行，可以在路由器后台查询树莓派的ip地址，但也可以使用以下方式

    1

    sudo ssh pi@raspberrypi.local

    `` 默认账号为pi，默认密码为raspberry，登录后系统会提示你修改

  2. 开启VNC（Virtual Network Computing），之后你就可以远程通过图形界面使用树莓派了

    1

    sudo raspi-config

  3. 选择Interfacing Options，选择VNC，重启

    1

    sudo reboot

  4. 在你的电脑上下载VNCViewer，登录方式与SSH类似

  5. 开始折腾！

可安装的其他系统

在自己电脑上折腾Linux系统很容易gg，但在树莓派上就非常方便，可以多准备几个SD卡换着用，但适用于树莓派的系统不多。

  • ArchLinux

    绝配，妙不可言，推荐使用i3桌面管理器，精简高效

  • Ubuntu Mate

    图形界面很好用，1G内存稍有不足

  • Windows 10 IoT Core

    主要面向物联网开发

  • RetroPie

    严格来说并不算是系统，它是一个游戏机模拟器，在上面你可以玩到非常多的复古游戏，推荐直接下载拷满游戏的镜像，为此你需要准备一个相当大的SD卡和一个复古手柄（强烈推荐八位堂8Bitdo），也可以使用PICO-8等复古游戏引擎自己开发游戏

  • Android

    没试过

  • OSMC

    家庭娱乐中心，也就是机顶盒

  • RISC OS

    硬核

其他用途

  • 日常学习

    用来学Python再适合不过

  • 机器人控制

    你可以在树莓派上安装摄像头，氢气球和传感器，让它飞向地球的边缘，去探索太空。当然了，这么做的话风险很大，一定要注意安全。这里指的不是你和你的树莓派，而是天上的飞机和住在下面的人们

  • 路由器

    openwrt需要交叉编译，但用其他方式也行，弃坑了

  • 网络附属存储（NAS）

    3b型号的网口与USB共享300Mbps的带宽，4b升级成了千兆网口和USB3.0，大有一番作为

  • 私有云

    ownCloud，度盘药丸

  • 网站服务器

    搭建个人博客，如WordPress（太臃肿）

  • P2P文件下载机Torrent

    听起来不错，待实现

  • 计算机集群

    使用多个树莓派搭建分布式系统，前提是要买得起
