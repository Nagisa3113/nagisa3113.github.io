花火大会

{% asset_img IMG_5785.jpeg %}

去的时候坐电车跟国内春运差不多，但人们素质都很高，路上也超多人，提前半小时已经没位置了

开始了嗷 {% asset_img IMG_5829.jpeg %} {% asset_img IMG_6175.jpg %}

整个过程会持续一个半小时左右，烟花表演因地区而各不相同，我们坐在河岸边的草地上，很多日本人会穿和服，带上零食饮料和家人一起来看，烟花在船上发射，种类很多，非常绚丽，整片天空都被点亮了
