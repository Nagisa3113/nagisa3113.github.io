了解世界

2020-03-02

文章目录

最近看了一篇文章王垠：完全用Linux工作

其中有些想法很偏激，但我很认可

Unix 是简单的，你不需要成为一个天才或是计算机专家。但是在这个冲斥着 Windows 错误观念的世界，你需要信念和勇气才能理解它的简单

后来又看了他的文章，很深刻清华梦的粉碎—写给清华大学的退学申请

还有许多内容在王垠的博客（不包括上两篇文章）

学习计算机有几年了，在这个过程中，发现它所涵盖的知识太多了，学校里教的完全不够，但这个过程是充满乐趣的，除了技术上的内容，也开始思考一些其他事情

正如学习语言一样，每一种语言都是一种全新的思维方式，如果只限于平时阅读交流，不去想语言背后隐藏的哲学，就无法真正理解语言。

也就是说

  • 编程是一种哲学

尽管会有Vim和Emacs这种类似宗教信仰的分支，但文章里是这么说的

当然我不是在这里打击 Linux 和 Mac 而鼓吹 Windows。这些系统的纷争基本上已经不关我什么事。我只是想告诉新人们，去除头脑里的宗教，偏激，仇恨和鄙视。每次仇恨一个东西，你就失去了向它学习的机会。

但国内的互联网环境太差了，以往编程高手都是广泛涉猎，接近真理的人，现在的人要么视野狭窄，活在自己的世界里，要么就是半吊子，了解一点皮毛就出来炫耀，真正厉害的程序员应该是谦逊的、友好的、不断进步的，导致这样的罪魁祸首是百度，这里懒得再去骂它了，另外cnblog和知乎也不是什么好东西（不谈商业属性）见知乎和 MITBBS，一个优秀的程序员不会浪费时间在这种虚幻的空间里。

最近还看了一本书《黑客与画家》，虽然它大部分在鼓吹创业和Lisp

其中谈到

  • 使用计算机以及其所有有助于了解这个世界本质的事物都不应受到任何限制。任何事情都应该亲手尝试
  • 信息应该免费
  • 计算机使生活更加美好

也就是说

  • 计算机是了解世界的方式

按理来说，程序员比普通人更容易接触到最新的/客观的信息，他们应该是更接近真理的人，应该更加谦卑，但遗憾的是很多人仅仅把计算机当作工具，或是沉迷在某些小圈子不可自拔，非常令人遗憾。

每一个学习计算机的人，不管你从事的是哪个开发方向，都会经历这些过程（虽然这话从我这个萌新口中说出来不太合适）

  • 精通英语
  • 掌握Linux
  • 有自己的服务器&技术博客
  • 了解各种编程语言/范式
  • 了解硬件知识，会组装各种设备
  • 在某些技术论坛上发表看法/指导新人
  • 为某个开源项目做出贡献
  • 配置属于自己的开发环境（物理/虚拟）
  • 通过计算机改善自己/周围人的生活质量
  • 通过客观/及时的消息了解自己生活的这个世界
  • 支持正版、认可别人的付出
  • 终生学习

还有很长的路要走呀
